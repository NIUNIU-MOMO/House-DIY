只输出 JSON，不要 markdown 代码块或额外说明。

输出格式：
{
  "rooms": [
    {
      "id": "r1",
      "polygon": [
        {"x": 120, "y": 95},
        {"x": 420, "y": 95},
        {"x": 420, "y": 310},

  ]

规则：
- 坐标单位为像素，原点在图片左上角；顶点 x∈[0,宽]，y∈[0,高]

- polygon 贴墙体围合内侧，顺时针；L 型/不规则房间用 4–8 个顶点
- 相邻房间共享墙体，polygon **禁止大面积重叠**
- 跟墙体线走，不要跟地板颜色块、家具外轮廓走
- 忽略中央水印/Logo、家具图标
