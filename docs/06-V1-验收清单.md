## 1. 主流程（01 → 08）

使用 **标准户型样本图**（建议 3 套不同布局）逐套执行：

- [ ] **01 项目列表** — 首页服务状态灯显示 oMLX / ComfyUI / Vault
- [ ] **02 上传** — 选择 PNG/JPG/PDF，填写预估面积，上传成功
- [ ] **03 解析** — VLM 解析进度可见，完成后进入校对（或失败可重试）
- [ ] **04 2D 校对** — 墙线/房间叠加显示，可改房间名，「确认户型，进入设计」成功
- [ ] **05 设计描述** — RAG 推荐案例列表加载，填写 brief + 风格，提交生成
- [ ] **06 生成进度** — 四步流水线（DesignSpec → 2D → 3D → Obsidian）进度更新
- [ ] **08 交付总览** — 显示项目名、房间列表、跳转画廊/3D/微调入口

**通过标准：** 至少 1 套样本全流程无阻塞错误；房间数与样本 ±1 可接受。

## 2. ComfyUI 效果图（V1 核心）

- [ ] **客厅** — `Cases/` 或画廊中有 ≥1 张效果图
- [ ] **主卧** — 同上
- [ ] **厨房** — 同上

验证路径：

- 前端：`/projects/:id/gallery`
- Vault：`~/House-DIY-Vault/Cases/<项目>/renders/`
- API：`GET /api/v1/projects/{id}/renders`

**通过标准：** 三房间均有 PNG；单张生成时间视 GPU，通常 < 15 min/张。

## 3. 3D 全屋漫游（10）

- [ ] **单房间** — WASD 移动，不穿墙
- [ ] **多房间** — ≥3 房间经门洞（Portal）切换
