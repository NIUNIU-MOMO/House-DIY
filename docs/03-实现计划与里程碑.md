# House-DIY 实现计划与里程碑

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 交付本地离线 Web 应用：标准户型图 → 校对 → 描述生成 DesignSpec → ComfyUI 2D + 程序化全屋 3D → Obsidian 案例沉淀与 RAG 复用。

**Architecture:** Vue3 + FastAPI 编排；oMLX（LLM/VLM/Embed）；ComfyUI 2D；Scene Builder + Three.js 3D；Obsidian Vault + Chroma RAG。

**Tech Stack:** Vue 3, FastAPI, SQLite, oMLX, ComfyUI, Three.js, ChromaDB, Obsidian（文件）, OpenCV, Shapely, trimesh

---

## 阶段总览

| 阶段 | 周期（估） | 交付物 |
|------|------------|--------|
| P0 基础设施 | 1–2 周 | 环境脚本、健康检查、空壳前后端 |
