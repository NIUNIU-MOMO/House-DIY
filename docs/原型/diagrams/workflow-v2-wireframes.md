```
  [1 上传]──[2 解析]──[3 标注]──[4 设计]──[5 预览]
     🟢        🟢        🟡        ⚪        ⚪
   已完成     已完成     进行中    未开始    未开始
```

| 样式类 | 含义 | CSS 参考 |
|--------|------|----------|
| `.step-done` | 已完成 | 绿 `#8fd4a8` / `#2c4a3e` |
| `.step-active` | 当前 | 黄 `#f0c14b` / `#5c4a1e` |
| `.step-pending` | 未到达 | 灰 `#888` / `#333`，disabled |

## 8. 状态机 — max_step

```mermaid
stateDiagram-v2
  [*] --> upload
  upload --> parse: 上传完成
  parse --> annotate: 解析成功
  annotate --> design: 保存标注 + 生成2D
  design --> preview: 3D就绪

  note right of design
    用户回到 annotate 编辑
    max_step 仍为 design
  end note
```
