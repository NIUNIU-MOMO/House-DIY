## 页面清单

| ID | 页面 | 演示要点 |
|----|------|----------|
| 00 | 项目列表 | 进入项目 → 最高进度步骤 |

## 可交互行为

- 流程条：**绿**（已完成）/ **黄**（当前）/ **灰**（未到达）
- 标注页点击「删除」或画布 → 标记 dirty → 切换步骤弹出**未保存提醒**
- 上传区点击 → 模拟选文件 → 启用「开始解析」
- 「放大预览」→ 原图/标注图层勾选弹窗
